# 早朝打卡 (Dawn Assembly)

> 三省六部每日晨会签到系统

## 项目简介

早朝打卡是一个用于管理团队每日晨会签到和汇报的系统。

## 功能特性

- 📋 每日签到打卡
- 📊 签到统计与报表
- 🔔 自动提醒通知
- 📱 多端同步支持

## 技术栈

- 后端：OpenClaw Agent
- 前端：待定
- 数据库：待定
